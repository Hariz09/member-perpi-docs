# Panduan Event PERPI

Panduan visual untuk mendaftar dan mengikuti event-event yang diselenggarakan oleh PERPI. Dokumentasi ini mencakup langkah-langkah registrasi event, pembelian tiket, melakukan konfirmasi pembayaran, hingga mendapatkan tiket event.

**Selamat mengikuti event PERPI! 🎉**
