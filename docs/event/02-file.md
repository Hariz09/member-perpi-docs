# Panduan Akses Direktori Dokumen

Panduan ini menjelaskan cara mengakses, menyaring, dan mengunduh berbagai berkas serta dokumen terkait event (seperti rundown, buku panduan, materi presentasi, atau dokumen kontrak) yang diselenggarakan oleh PERPI.

:::info Info Dokumen
Sistem member membatasi akses berkas berdasarkan status pendaftaran dan peran (role) Anda. Pastikan Anda masuk menggunakan akun yang sesuai untuk melihat dokumen terbatas.
:::

---

## Cara Mengakses Dokumen

Anda dapat melihat dan mengunduh berkas event melalui **dua metode** berikut:

### Metode 1 — Melalui Sidebar "Direktori Dokumen" (Global)

Metode ini digunakan untuk melihat seluruh dokumen dari berbagai event yang Anda ikuti secara terpusat dalam satu halaman.

**1.** Pada menu navigasi utama (sidebar) sebelah kiri, gulir ke bagian **Event**.
**2.** Klik menu **Direktori Dokumen**.

<img src="/img/event/02-file/1-sidebar-direktori.gif" alt="Menu Direktori Dokumen pada Sidebar" className="img-large" />

**3.** Anda akan diarahkan ke halaman daftar berkas global. Halaman ini memuat seluruh file dari event-event aktif yang memiliki hak akses untuk akun Anda.

---

### Metode 2 — Melalui Halaman Detail Event (Spesifik)

Jika Anda sedang melihat informasi event tertentu dan ingin mencari berkas khusus event tersebut, gunakan metode ini.

**1.** Buka menu **Daftar Event** di sidebar.
**2.** Pilih salah satu event yang ingin Anda lihat detailnya.
**3.** Pada halaman detail event, klik tab **Materi / Dokumen Event** (atau **File Event**).

<img src="/img/event/02-file/2-detail-event-files.gif" alt="Tab File Event pada Detail Event" className="img-large" />

**4.** Halaman ini hanya akan menampilkan berkas-berkas yang relevan dengan event terpilih tersebut.

---

## Fitur & Elemen Tabel Dokumen

Halaman Direktori Dokumen dilengkapi dengan beberapa fitur untuk memudahkan pengelolaan berkas:

<img src="/img/event/02-file/3-tabel-direktori-fitur.png" alt="Fitur Tabel Direktori Dokumen" className="img-large" />

### 1. Pencarian & Filter Instan
Gunakan kolom pencarian di bagian atas tabel untuk menyaring berkas berdasarkan:
- Nama berkas (contoh: `Rundown`, `Slide`)
- Deskripsi singkat berkas
- Nama Event yang bersangkutan

### 2. Informasi Detail Kolom
Tabel ini menampilkan detail informasi sebagai berikut:
- **File**: Nama berkas beserta deskripsi singkat dan badge jenis file (contoh: *Rundown*, *Guidelines*, *Presentation*, *Contract*, *Other*).
- **Event**: Nama event yang terkait dengan berkas tersebut.
- **Akses**: Tingkat keamanan/visibilitas dokumen (lihat penjelasan tingkat akses di bawah).
- **Ukuran**: Ukuran berkas dalam format byte (B), Kilobyte (KB), atau Megabyte (MB).
- **Tanggal**: Tanggal berkas tersebut diunggah oleh panitia.

### 3. Ekspor Data
Anda dapat mengekspor daftar berkas yang tampil menjadi file spreadsheet dengan menekan tombol **"Export"** di bagian kanan atas tabel.

---

## Cara Mengunduh Berkas

Untuk menyimpan berkas ke perangkat Anda:

**1.** Temukan berkas yang ingin diunduh pada tabel.
**2.** Klik tombol ikon **Download** (ikon panah bawah) di kolom paling kanan tindakan.
**3.** Berkas akan langsung terunduh secara otomatis melalui browser Anda.

:::warning Gagal Mengunduh?
Jika muncul pesan kesalahan *"Gagal mengunduh dokumen"*, pastikan koneksi internet Anda stabil atau hubungi pihak sekretariat karena ada kemungkinan hak akses berkas telah diubah oleh admin.
:::

## Memahami Tingkat Akses (Visibilitas) Berkas

Setiap berkas memiliki tingkat visibilitas berbeda yang ditentukan oleh Panitia/Event Manager:

| Tingkat Akses | Ikon | Deskripsi |
| :--- | :---: | :--- |
| **Publik** | 👥 | Dapat diakses dan diunduh oleh siapa saja (termasuk Guest/Tamu). |
| **Peserta** | 👥 | Hanya dapat diakses oleh pengguna yang sudah terdaftar secara resmi pada event terkait. |
| **Terbatas** | 🔒 | Hanya dapat diakses oleh peran (role) tertentu (seperti *Ketua, Panitia, Keuangan*), kategori slot tertentu, atau email pengguna yang secara spesifik masuk daftar putih (*whitelist*). |

:::tip Detail Aturan Akses
Arahkan kursor Anda ke badge status **Akses** (khusus untuk berkas berstatus **Terbatas**) untuk melihat daftar peran atau aturan spesifik yang diperbolehkan mengakses dokumen tersebut melalui *tooltip* bantuan.
:::
